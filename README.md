# 📝 Task Manager - Danna Villegas

## Descripción
Aplicación web desarrollada para gestionar tareas de forma sencilla, permitiendo agregar, organizar y eliminar tareas con prioridad y fecha de vencimiento. 
Este proyecto fue creado con el objetivo de fortalecer habilidades en desarrollo web frontend.

---

## Tecnologías
- HTML5
- CSS3
- JavaScript
- Git y GitHub
- Responsive Design

---

## Funcionalidades
- Agregar nuevas tareas
- Seleccionar prioridad (Alta, Media, Baja)
- Asignar fecha límite a cada tarea
- Eliminar tareas del listado
- Interfaz dinámica e interactiva
